‚ö° The Multiplicative Rule

Core Concept

Calculate the probability of both events occurring (intersection) using conditional probability.

Multiplicative Rule Formula

$$P(A \cap B) = P(A) \cdot P(B \mid A) = P(B) \cdot P(A \mid B)$$

This comes from rearranging the conditional probability formula!

üí° Key Insight

Translation: "Probability of both" = "Probability of first" √ó "Probability of second given first occurred"

üî¨ Scientific Application: Milgram Psychology Study

Research Context

Stanley Milgram's famous 1960s obedience experiments studied teacher behavior when asked to shock learners for wrong answers.

Events of Interest:

  • Event A: Teacher applies severe shock (450 volts)
  • Event B: Learner protests verbally before shock

Research Data: P(B) = 0.5, P(A | B) = 0.7

üìà Applying the Multiplicative Rule

1
Research Question:
What's P(learner protests AND teacher applies severe shock)?
2
Apply Multiplicative Rule:
P(A ‚à© B) = P(B) √ó P(A | B)
P(A ‚à© B) = 0.5 √ó 0.7 = 0.35
3
Interpretation:
35% of cases involve both verbal protest AND severe shock application

üö® Key Scientific Finding

35% of interactions show disturbing obedience: Even when learners protest, teachers still apply severe shocks over one-third of the time.

This quantifies the psychological tendency to obey authority figures despite moral concerns.

üéØ Independence: The Complete Picture

Independence Definition

Events A and B are independent if:

$$P(A \mid B) = P(A) \text{ and } P(B \mid A) = P(B)$$

Equivalent test:

$$P(A \cap B) = P(A) \times P(B)$$

‚úÖ Independent Events

Examples:

  • Coin tosses
  • Card color & face status
  • Medical test results (different patients)
  • Rolling dice multiple times
Formula: P(A ‚à© B) = P(A) √ó P(B)

‚ùå Dependent Events

Examples:

  • Drawing cards without replacement
  • Weather on consecutive days
  • Sports "hot" streaks
  • Stock prices over time
Formula: P(A ‚à© B) = P(A) √ó P(B | A)

‚ö†Ô∏è Counter-Example: Dependent Events

Drawing Without Replacement

Consider drawing two cards WITHOUT replacing the first:

  • Event A: First card is an Ace
  • Event B: Second card is an Ace

üîç Dependence Analysis

1
P(A): P(first Ace) = 4/52 = 0.0769
2
P(B | A): P(second Ace | first was Ace) = 3/51 = 0.0588
(3 Aces left in 51 remaining cards)
3
P(B | A^c): P(second Ace | first not Ace) = 4/51 = 0.0784
(4 Aces left in 51 remaining cards)

‚ùå Clear Dependence

The first draw affects the second!

P(B | A) = 0.0588 ‚â† P(B | A^c) = 0.0784

Different conditional probabilities prove dependence

üö´ Common Mistakes to Avoid

‚ö†Ô∏è Don't Confuse These Concepts:

  • Independent ‚â† Mutually Exclusive: Independent events CAN occur together
  • P(A ‚à© B) ‚â† P(A | B): Intersection vs conditional probability
  • P(A | B) ‚â† P(B | A): Order matters in conditional probability
  • Independence ‚â† Causation: Independence doesn't imply causation

‚úÖ Problem-Solving Strategy

1
Clearly define both events
2
Check if independence is given or must be tested
3
For independent: use P(A ‚à© B) = P(A) √ó P(B)
4
For dependent: use P(A ‚à© B) = P(A) √ó P(B | A)
5
Always interpret results in context

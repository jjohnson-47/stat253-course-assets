Section 3.4: The Additive Rule and Mutually Exclusive Events

üéØ The Big Picture: Adding Probabilities

When we want to find the probability that either event A or event B occurs (or both), we need to use the Additive Rule. But there's a catch‚Äîwe need to be careful about double-counting!

üìä General Additive Rule

For any two events A and B:

$$P(A \cup B) = P(A) + P(B) - P(A \cap B)$$

We subtract the intersection to avoid counting it twice!

‚ú® Special Case: Mutually Exclusive

When A and B cannot happen together:

$$P(A \cup B) = P(A) + P(B)$$

Simple addition because there's no overlap!

üîç Understanding Mutually Exclusive Events

Mutually Exclusive Events

Definition: Two events A and B are mutually exclusive if they cannot occur simultaneously. In other words, $A \cap B = \emptyset$ (the empty set).

Key insight: If events are mutually exclusive, then $P(A \cap B) = 0$

Overlapping Events

NOT Mutually Exclusive:
$P(A \cap B) > 0$

Mutually Exclusive Events

Mutually Exclusive:
$P(A \cap B) = 0$

üè• Real-World Example: Hospital Admissions

The Scenario

Hospital records show that of all patients admitted:

  • 12% are admitted for surgical treatment
  • 16% are admitted for obstetrics
  • 2% receive both surgical and obstetrics treatment

Question: What's the probability that a new patient will be admitted for surgery, obstetrics, or both?

üìù Step-by-Step Solution

1 Define the events:
A = "Patient receives surgical treatment"
B = "Patient receives obstetrics treatment"
2 Identify the given probabilities:
$P(A) = 0.12$, $P(B) = 0.16$, $P(A \cap B) = 0.02$
3 Apply the general additive rule:
$P(A \cup B) = P(A) + P(B) - P(A \cap B)$
4 Calculate:
$P(A \cup B) = 0.12 + 0.16 - 0.02 = 0.26$

Answer: 26% of patients receive surgery, obstetrics, or both

ü™ô Mutually Exclusive Example: Coin Tosses

The Experiment

Toss two balanced coins. Find the probability of observing at least one head.

Sample Space: $S = \{HH, HT, TH, TT\}$

üß† Strategic Solution Using Mutually Exclusive Events

1 Define helpful events:
A = "Observe at least one head"
B = "Observe exactly one head" = $\{HT, TH\}$
C = "Observe exactly two heads" = $\{HH\}$
2 Key insight: $A = B \cup C$ and B and C are mutually exclusive
(You can't have exactly one head AND exactly two heads simultaneously!)
3 Calculate individual probabilities:
$P(B) = P(\{HT, TH\}) = \frac{2}{4} = \frac{1}{2}$
$P(C) = P(\{HH\}) = \frac{1}{4}$
4 Apply mutually exclusive rule:
$P(A) = P(B \cup C) = P(B) + P(C) = \frac{1}{2} + \frac{1}{4} = \frac{3}{4}$

Answer: $\frac{3}{4}$ or 75% chance of at least one head

CAUTION: Common Mistake

The simple addition rule $P(A \cup B) = P(A) + P(B)$ is ONLY valid for mutually exclusive events!

If events can overlap, you must subtract the intersection: $P(A \cup B) = P(A) + P(B) - P(A \cap B)$

Example of the mistake: If you wrongly calculated the hospital example as $P(A \cup B) = 0.12 + 0.16 = 0.28$, you'd be double-counting the 2% who receive both treatments!

üß© Quick Recognition Guide

üîç How to Identify Mutually Exclusive Events

Ask yourself: "Can these events happen at the same time?"

Examples of mutually exclusive:

  • Rolling a 3 OR rolling a 5 on one die
  • Being exactly 20 years old OR being exactly 25 years old
  • A student being a freshman OR being a senior

‚ö° When Events Are NOT Mutually Exclusive

These CAN happen together:

  • Being tall AND being athletic
  • Owning a car AND owning a bike
  • Getting an A in math AND getting an A in English
  • The hospital surgery/obstetrics example

‚úÖ Key Takeaways

Master These Concepts:

  1. General Additive Rule: $P(A \cup B) = P(A) + P(B) - P(A \cap B)$ (always works)
  2. Mutually Exclusive Rule: $P(A \cup B) = P(A) + P(B)$ (only when $A \cap B = \emptyset$)
  3. Recognition: Ask "Can these events happen simultaneously?"
  4. Strategy: Breaking "at least" problems into mutually exclusive pieces
  5. Caution: Never use simple addition unless events are mutually exclusive

The Decision Tree

Step 1: Are the events mutually exclusive?

If YES: Use $P(A \cup B) = P(A) + P(B)$

If NO: Use $P(A \cup B) = P(A) + P(B) - P(A \cap B)$

If UNSURE: Always use the general rule‚Äîit works for both cases!
